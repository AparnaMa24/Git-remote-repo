Getting a Git Repository:

Step1 : create a folder on desktop/in any drive with name gitlocalrepository3rdapr
Step2 : add 1 text file and 1 word document with no content
Step3 : convert the folder into local repository using below command
         > git init

Step 4: check the status of git local repo
       >git status
          some untracked files are listed in red color
Step 5: track the files by staging them
       >git add .

Step 6: check the status of git local repo
       >git status
         files are tracked(in staging area) but yet to be commited
Step 7: commit the files into local git repo
       > git commit -m " empty text and doc files are added"
        files are commited into the local repo
Step 8: check the status of git local repo
       >git status
        On branch master
        Nothing to commit, working tree is clean
Step 9 : update text and word doc in working directory adding a single line of text through windows explorer
Step 10: check the status of git local repo
       >git status
         modified 2 files yet to be staged
Step 11: stage modified files
       >git add .
Step 12: check the status of git local repo
       >git status
       2 modifies files are staged yet to be commited
Step 13: commit modified files
      >git commit -m "both the files are added with some content"
       files are commited into the local repo
Step 14: check the status of git local repo
       >git status
        On branch master
        Nothing to commit, working tree is clean
